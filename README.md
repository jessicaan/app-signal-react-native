# Signal-Clone

This project was created to replicate the signal app and some of its functions (in a simpler way).
Created using react-native.

## Working functions

Is possible to sign in and log in using e-mail and password authentication, send and receive messages from other user.
P.s.: The hours and texts are in PT-BR pattern.
